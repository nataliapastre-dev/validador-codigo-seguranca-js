# Validador de Código de Segurança

Projeto em JavaScript que valida códigos de segurança com regras específicas.

## Objetivo
Verificar se um código atende regras de formatação antes de liberar acesso.

## Regras
Um código é válido quando:
- Possui exatamente 8 caracteres
- Começa com a letra **S**
- Termina com a letra **X**
- Contém exatamente 3 números no meio

## Tecnologias
- JavaScript

## Lógica aplicada
- Manipulação de strings
- Laço de repetição
- Condições lógicas

## Exemplo

Entrada:
SA1B2C3X

Saída:
ACESSO LIBERADO
